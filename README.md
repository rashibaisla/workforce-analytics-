# Workforce Analytics Dashboard

A modern Flask-based Workforce Analytics System designed to manage and visualize workforce-related data through a clean and responsive dashboard interface.

This project demonstrates full-stack fundamentals including backend routing, frontend templating, and deployment of a live web application.

🚀 Live Demo

👉 https://workforce-analytics-1yw2.onrender.com/

📌 Key Features
🔐 User Authentication System (Login / Register)
📊 Interactive Dashboard Interface
🎯 Modular UI using HTML Templates
🎨 Clean and modern CSS-based design
🌐 Fully deployed web application
⚡ Lightweight and fast Flask backend

🛠️ Tech Stack:

-Backend-
Python
Flask

-Frontend-
HTML5
CSS3
Jinja2 Templates (Flask templating engine)

-Deployment-
Render (Cloud Hosting)
GitHub (Version Control)


📁 Project Structure:

workforce-analytics/
│
├── app.py                  # Main Flask application
├── requirements.txt        # Project dependencies
│
├── templates/              # HTML pages
│   ├── dashboard.html
│   ├── login.html
│   └── register.html
│
├── static/                 # CSS and assets
│   └── style.css
│
└── venv/                   # Virtual environment (not deployed)


⚙️ How It Works:

User registers or logs in
Authentication handled via Flask routes
Dashboard loads after login
Data displayed in structured UI cards
Backend serves templates dynamically using Jinja2

📦 Installation (For Local Setup):
git clone https://github.com/your-username/workforce-analytics.git
cd workforce-analytics

pip install -r requirements.txt

python app.py

Then open:

http://127.0.0.1:5000
